AKITAmobile について

　AKITAmobile（以下，「本アプリ」といいます。）は、秋田県に関する情報を発信するインフルエンサー、観光関連業者、自治体などのWEBページ,SNSのリンクをまとめて閲覧できるサービスです。

　登録ユーザーの皆さま（以下，「ユーザー」といいます。）はご自身のWEBページ,SNSのリンクを投稿することができ、他のユーザーのリンクをご覧になれます。

　外部リンクの内容等について、AKITAmobileが、内容の真実性や有効性を保証したり推薦したりするものではありません。また、それぞれのリンク先は、AKITAmobileの管理下ではないため、内容やそれらを使用したことによる損害について、AKITAmobileに故意又は重過失がある場合を除き、責任を負いません。

　本アプリの開発および運営にあたり、開発者はApple Developer Program使用許諾契約（Apple Developer Program License Agreement）を含む、Appleが定める各種ガイドラインおよびポリシーを遵守しております。

※本アプリは秋田県の公式アプリではなく、非公式アプリです。
